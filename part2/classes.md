# Part 2 #

**class**
  - door

**attributes**

  - is_open (boolean)
  - is_clean (boolean)
  - height (int)
  - width (int)
  - color (string)

**Methods**
  - close_door (sets "is_open" to either true or false)
  - clean_door (sets "is_clean" to either true or false)
  - change_color (changes color or the door)
  - calculate_sq_footage (height * width)
